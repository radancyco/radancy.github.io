TalentBrew Admin Plus

An interim solution to dealing with some of the shortcomings of the Site Admin tool until proper enhancements are in place.

Instructions

Drag or right-click and add bookmarklet to your browser's bookmarks folder or bar, go to admin tool, navigate to desired section and access saved bookmark. Magic.

Themes & Pages

Themes & Pages: Append URL

Appends clickable page URL to each row in the list of pages within a theme.

Themes & Pages: Highlight Page Types

Highlight groups of page types in each theme for easier viewing.

Themes & Pages: Sort Pages By Name

Sorts all pages within each theme, allows page name to be opened in new window.
