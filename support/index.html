Web Standards

Our development methodologies adhere to many standards and guidelines proposed by the W3C.

We also embrace best practices found in the web design and development community. By using the most modern techniques at our disposal, we can craft an experience that is both engaging and accessible to all users.

By adopting modern principles of design and development, we can also ensure stability. Web sites that work well in today's modern browsers will also work well in future browsers. This is the promise of supporting standards based development and best practices.

To learn more about our open coding standards, please visit us on GitHub.

Accessibility

All of your prospective job candidates, regardless of physical or technological capabilities, have the right to learn more about your company and access all job listings. To preserve users' right of universal access, our baseline development standards for all of our web offerings attempt to meet as many WCAG 2.1 Level A and AA guidelines as possible. However, many aspects of accessibility touch upon how a site is designed and implemented and require a more stringent process, so if there is a need to fully support accessibility guidelines, please bring this to the attention of your account representative as soon as possible.

Operating System Support

TMP supports applications on the following operating systems:

  • Windows 10
  • Windows 8/8.1
  • macOS 10.14 (Mojave)
  • iOS 12.x
  • iOS 11.x
  • Android 8.x
  • Android 7.x
  • Android 6.x

Browser Support

With the exception of Internet Explorer, TMP supports the current and prior major release versions of all browsers. Corresponding to the date in which your project has launched, we test all development in the following:

  • Google Chrome
  • Mozilla Firefox
  • Apple Safari
  • Microsoft Edge
  • Internet Explorer 11

Email Support

Corresponding to the date in which your project has launched, we test all text and HTML based emails in the following email clients:

  • Outlook (Latest version of Outlook on Windows 10 only)
  • iPhone (Latest version of iOS Mail only)
  • Gmail Desktop (Latest browser versions only, except for Internet Explorer)
  • Gmail Android (Upon Request Only)

Please note that we do not support browsers that have been modified, run in compatibility mode, or filter out code via proxy filtering, that is required for the normal operation of your site.

Our development team analyzes global browser market share and client web traffic data on a quarterly basis. Web browser support ends when that browser represents less than 5% of all market share and/or web traffic.

We do not, as a matter of course, support browsers that are not listed within this document. Should your users or company's browser needs differ, please consult your account representative immediately.
